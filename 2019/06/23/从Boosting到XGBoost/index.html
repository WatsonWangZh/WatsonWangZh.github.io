前言

   集成学习在各种数据科学比赛中都有举足轻重的地位，而基于树模型的XGBoost更是其中的常客。XGBoost 是 “Extreme Gradient Boosting”的简称,以CART树作为基础模型实现。
   本文将按的顺序介绍,以期对XGBoost有比较深入的认识。
